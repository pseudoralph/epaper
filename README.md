## Huzzah Feather v2 && Waveshare 4.2v2 in e-paper

[based on](https://files.waveshare.com/upload/7/71/E-Paper_code.zip)

#### Pinout

```
WAVESHARE                   FEATHER
---------------------------------------------
VCC           (gry)         3.3V
GND           (br)          GND
DIN           (bl)          MOSI    (GPIO19)
CLK/SCLK      (yl)          SCK     (GPIO5)
CS            (or)          A8      (GPIO15)
DC            (grn)         A10     (GPIO27)
RST           (wh)          A0      (GPIO26)
BUSY          (pur)         A1      (GPIO25)
```
